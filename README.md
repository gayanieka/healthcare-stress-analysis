# Healthcare Stress Analysis

## Overview
This project analyzes whether positive healthcare experiences reduce patient stress levels using real-world data.

## Objective
To test the statement:
"Positive healthcare experiences reduce patient stress levels"

## Methods
- Descriptive Analysis
- Inferential Analysis (Shapiro-Wilk, t-test, Spearman correlation)
- Predictive Analysis (Multiple Linear Regression)

## Key Findings
- Stress levels were generally low
- Data was not normally distributed
- Depression had the strongest effect on stress
- Medical cost was not significant
- Model explained 14.7% of stress variation

## Conclusion
The statement is not strongly supported by the analysis.

- R Programming
- Statistical Modelling
